# 项目介绍

基于 uni-app + wot-design-uni + Vue 3 + TypeScript 移动端跨平台开发模板，集成了 ESLint、Prettier、Stylelint、Husky 和 Commitlint 等工具，确保代码规范与质量。

## 项目启动

安装依赖

```node
pnpm install
```

h5启动

```node
pnpm run dev:h5
```

构建

```node
pnpm run build:h5
```

访问 [http://localhost:5180/app](http://localhost:5180/app)
